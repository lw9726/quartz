# 001-诗-将进酒

**作者:** [[诗人/001-诗人-李白]]  
**朝代:** [[时空/001-朝代-唐朝]]  
**体裁:** 乐府诗  
**创作时间:** 约天宝十一载（752年）

**原文:**
```
君不见，黄河之水天上来，奔流到海不复回。
君不见，高堂明镜悲白发，朝如青丝暮成雪。
人生得意须尽欢，莫使金樽空对月。
天生我材必有用，千金散尽还复来。
烹羊宰牛且为乐，会须一饮三百杯。
岑夫子，丹丘生，将进酒，杯莫停。
与君歌一曲，请君为我倾耳听。
钟鼓馔玉不足贵，但愿长醉不复醒。
古来圣贤皆寂寞，惟有饮者留其名。
陈王昔时宴平乐，斗酒十千恣欢谑。
主人何为言少钱，径须沽取对君酌。
五花马，千金裘，呼儿将出换美酒，与尔同销万古愁。
```

**创作背景:**
此诗作于[[诗人/001-诗人-李白]]被唐玄宗"赐金放还"离开长安后。诗人在[[时空/010-地点-嵩山]]与友人[[诗人/008-诗人-岑勋]]、[[诗人/009-诗人-元丹丘]]聚会时所作，表达了对人生短暂的感慨和及时行乐的人生态度。

**主要意象:**
- [[概念/008-概念-黄河意象]]：象征时光流逝，人生短暂
- [[概念/009-概念-酒文化]]：表达豪放不羁的性格和对现实的超脱
- [[概念/010-概念-金钱观]]：体现诗人轻财重友、豁达洒脱的人生观

**艺术特色:**
1. **[[概念/011-概念-起兴手法]]**：以"君不见"起兴，气势磅礴
2. **[[概念/012-概念-对比手法]]**：时间对比（朝暮）、空间对比（天地）
3. **[[概念/013-概念-夸张手法]]**：一饮三百杯、斗酒十千等夸张描写
4. **[[概念/014-概念-用典手法]]**：陈王（曹植）典故的运用

**思想主题:**
- [[主题/004-主题-人生哲理]]：对人生短暂的深刻思考
- [[主题/005-主题-及时行乐]]：珍惜当下，把握人生
- [[主题/006-主题-友谊珍贵]]：与朋友共享欢乐时光
- [[主题/001-主题-个性解放]]：反抗世俗，追求精神自由

**后世影响:**
这首诗是[[诗人/001-诗人-李白]]的代表作之一，体现了盛唐诗歌的豪放风格，对后世诗歌创作产生了深远影响。其中"天生我材必有用"等名句成为激励人心的经典名言。
