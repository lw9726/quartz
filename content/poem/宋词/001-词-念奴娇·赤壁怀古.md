# 001-词-念奴娇·赤壁怀古

**作者:** [[诗人/003-诗人-苏轼]]  
**朝代:** [[时空/006-朝代-北宋]]  
**词牌:** 念奴娇  
**创作时间:** 元丰五年（1082年）

**原文:**
```
大江东去，浪淘尽，千古风流人物。
故垒西边，人道是，三国周郎赤壁。
乱石穿空，惊涛拍岸，卷起千堆雪。
江山如画，一时多少豪杰。

遥想公瑾当年，小乔初嫁了，雄姿英发。
羽扇纶巾，谈笑间，樯橹灰飞烟灭。
故国神游，多情应笑我，早生华发。
人生如梦，一尊还酹江月。
```

**创作背景:**
此词作于[[诗人/003-诗人-苏轼]]被贬[[时空/008-地点-黄州]]期间，游览赤壁时所作。虽然此赤壁并非[[时空/011-地点-三国赤壁]]真正的古战场，但诗人借古抒怀，表达了对历史英雄的仰慕和对人生的深刻思考。

**主要意象:**
- [[概念/015-概念-大江东去]]：象征历史的滚滚洪流
- [[概念/016-概念-风流人物]]：指历史上的英雄豪杰
- [[概念/017-概念-江山如画]]：自然美景与人文情怀的结合
- [[概念/018-概念-人生如梦]]：对人生短暂的哲理思考

**历史人物:**
- [[历史人物/001-人物-周瑜]]：三国时期东吴名将，字公瑾
- [[历史人物/002-人物-小乔]]：周瑜妻子，东汉末年美女
- [[时空/012-事件-赤壁之战]]：三国时期著名战役

**艺术特色:**
1. **[[概念/006-概念-豪放派]]**：气势恢宏，胸襟开阔
2. **[[概念/019-概念-咏史怀古]]**：通过历史人物抒发情感
3. **[[概念/020-概念-情景交融]]**：自然景色与历史感慨完美结合
4. **[[概念/021-概念-虚实结合]]**：现实景象与历史想象交织

**思想主题:**
- [[主题/007-主题-历史兴衰]]：对历史变迁的深刻思考
- [[主题/003-主题-人生哲理]]：人生如梦的哲学感悟
- [[主题/008-主题-英雄崇拜]]：对历史英雄的仰慕之情
- [[主题/009-主题-时光易逝]]：对时间流逝的感慨

**文学地位:**
这首词是[[概念/006-概念-豪放派]]词的代表作，开创了词的新境界。它将[[概念/019-概念-咏史怀古]]的传统主题与个人情感完美融合，体现了[[诗人/003-诗人-苏轼]]深厚的文学功底和人生智慧。

**名句赏析:**
- "大江东去，浪淘尽，千古风流人物"：气势磅礴的开篇，奠定了全词的豪放基调
- "人生如梦，一尊还酹江月"：哲理性的结尾，体现了作者的人生感悟
