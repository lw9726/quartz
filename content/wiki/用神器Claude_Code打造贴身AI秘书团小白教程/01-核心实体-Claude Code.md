Claude Code（简称CC）是Anthropic公司发布的编程代理，它不仅是强大的编程工具，更是一个可以深度个性化、辅助思考的思维搭子。
> OK，我们先花 30 秒快速认识一下今天的主角 Cloud Code。也有人呢简称它为 CC。 它呢是 Cloud 的母公司 Anthropic 今年 5 月正式发布的编程代理。
> 它最被低估的用法就是做你的思维搭子。

Claude Code是本教程的核心工具，被誉为编程领域的“Agent大哥大”。与普通的聊天AI不同，它被设计为一个能够深度了解用户、使用无限工具箱并能带领团队的“贴身秘书机器人”。其最关键的特性是[[003-核心实体-Sub Agent (子代理)|Sub Agent（子代理）]]功能，允许它根据复杂任务生成并调派一群具有不同特长的小秘书，从而组建一个[[02-核心实体-AI秘书团]]。教程建议通过[[04-核心实体-Cursor (AI IDE)]]来简化其[[008-核心流程-安装与配置Claude Code|安装和配置]]过程。
