MCP（外挂工具）是Claude Code可以调用的外部工具或插件，用于扩展其能力，如联网搜索、文件操作、调用API等。
> 你呢还能给它接入社区里那些成千上万的 MCP 外挂工具。
> 那像天气 MCP 啊，Fire Call，就是搜索和爬取的 MCP，我也都是在 Smithery 里面去挑了一下，操作都是一样的逻辑。

MCP是[[01-核心实体-Claude Code]]的“手和脚”，使其能够与外部世界交互，而不仅仅是进行文本对话。这些工具可以通过[[010-核心流程-配置MCP工具|专门的流程]]进行安装和配置。教程中提到了可以去[[006-核心实体-Smithy (工具超市)|Smithy]]工具超市挑选所需工具，例如用于生成穿搭参考图的“极梦”或用于发送消息的“飞书”。为不同的[[003-核心实体-Sub Agent (子代理)|Sub Agent]]配置不同的MCP，是实现其专业能力的关键。
