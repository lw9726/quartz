安装与配置Claude Code是使用该工具的第一步，教程推荐借助AI IDE来简化这个过程。
> 保证它是 Agent 的模式。然后呢，我们再把 Cloud Code 的这个官方文档说明给它发给他，对他说「帮我安装一个 Cloud Code」。
> 那装好之后，我们还得给它换个大脑...我们直接去智谱的模型平台...去获取一个 API Key 给它拷贝上。

本教程介绍了一种对新手极其友好的[[008-核心流程-安装与配置Claude Code|安装与配置方法]]。它利用[[004-核心实体-Cursor (AI IDE)|Cursor]]的Agent模式，让AI自己阅读官方文档并完成安装，避免了用户手动处理Node.js等环境依赖的麻烦。配置阶段则包括替换模型（将默认的Claude模型替换为GLM 4.5等国产模型）和设置API Key，从而让[[001-核心实体-Claude Code]]能在用户的环境中顺利运行。
