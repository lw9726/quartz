Sub Agent（子代理）是Claude Code的一项核心功能，允许一个主Agent根据任务需求生成并委派多个具有特定技能的子Agent。
> 它还能带团队。 一个大秘书啊，他可以根据你的需求来生成一群不同特长的小秘书。等到该谁上的时候呢，这个大秘书就去调派谁来上场。这就是 Cloud Code 最近更新的 Sub Agent 子代理功能。

Sub Agent功能是[[001-核心实体-Claude Code]]强大能力的核心，也是构建[[002-核心实体-AI秘书团]]的技术基础。它使得Claude Code不再是一个单体应用，而是一个可以动态创建和管理团队的“总秘书”。用户可以通过[[011-核心流程-创建Sub Agent|创建子代理]]的流程，为每个子代理定义其独特的角色、提示词和可使用的[[005-核心实体-MCP (外挂工具)|工具]]，从而实现复杂任务的分解和自动化处理。
