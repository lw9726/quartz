创建Sub Agent是利用Claude Code的内置命令，将设计好的秘书角色实例化为具体的AI智能体的过程。
> 我们输入 Cloud，进入到对话之后呢，直接 -agent。然后我们选 create，创建一个 agent。
> 好了，第一个新闻丘就这么诞生了，不难吧。

这是将[[009-核心流程-设计秘书团|设计蓝图]]变为现实的核心步骤。该流程通过在[[01-核心实体-Claude Code]]的对话界面中执行`/agent create`命令来完成。在创建过程中，系统会引导用户完成以下设置：
1.  确定Agent的作用范围（项目内或全局）。
2.  提供Agent的核心职责描述（一句话）。
3.  为Agent分配其可使用的[[005-核心实体-MCP (外挂工具)|工具]]。
4.  选择一个代表颜色。
完成这些步骤后，一个[[003-核心实体-Sub Agent (子代理)|子代理]]便被创建出来，其配置文件会出现在`.claude/agents`目录中，用户也可以后续手动修改。
