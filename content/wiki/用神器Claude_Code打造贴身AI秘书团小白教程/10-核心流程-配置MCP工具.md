配置MCP工具是为AI秘书团的成员“装备武器”的过程，使其能与外部世界交互。
> 那么设计好了，我们就来亲手孵化它们吧。这一步我们需要先给秘书们配置一些工具，也就是 MCP。
> 我们可以输入 Cloud MCP 的 list 来检查一下这些 MCP 都配置成功没有。

在[[011-核心流程-创建Sub Agent|创建Sub Agent]]之前，需要先为它们准备好将要使用的[[005-核心实体-MCP (外挂工具)|工具]]。这个流程包括：
1.  去[[006-核心实体-Smithy (工具超市)|Smithy]]寻找合适的工具。
2.  根据Smithy提供的命令和说明，在命令行中进行安装和配置，这通常需要提供API Key。
3.  教程还提供了利用AI（如官方文档的问答AI或通用大模型）来辅助理解和转换复杂配置命令的技巧。
4.  最后，通过` /mcp list `命令检查所有工具是否都已成功连接。
