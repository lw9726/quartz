Cloud Code是作者用于制作自动化工作流，以实现社群内容整理和分析的另一个工具。

> 我为社群内容做整理和分析，我就用Cloud Code制作了另一套工作流，能够自动抓取社群网站的数据，并转化为大模型可以读取的格式。

## 展开阐述

Cloud Code是作者实践[[1-核心概念-Vibe Coding]]中“自动化的工作流”的又一个实例。与使用[[14-核心实体-N8N]]和[[15-核心实体-Gemini CLI]]处理视频不同，这个工作流专注于数据处理和分析。

该工作流的核心功能是：
1.  自动抓取社群网站的数据。
2.  将抓取的数据转化为大模型（LLM）可以读取和理解的格式。

这个案例进一步展示了[[1-核心概念-Vibe Coding]]的广泛适用性。通过这种方式，作者能够实现过去“想都不敢想”的内容洞察，这体现了AI赋能个体进行深度数据分析的能力。它再次证明，核心能力是用清晰的语言向AI描述需求和步骤，而不是编程本身。
