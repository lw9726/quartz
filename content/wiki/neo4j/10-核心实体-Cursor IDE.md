Cursor IDE 是一个与模型上下文协议（MCP）兼容的客户端应用程序。
> There are many MCP-compatible clients, such as Claude Desktop and the Cursor IDE.

## 展开阐述
Cursor IDE 是一个集成了 AI 功能的集成开发环境（IDE），它与 [[001-核心概念-Model Context Protocol (MCP)|MCP]] 兼容，可以作为利用 MCP 服务器的客户端之一。

与 [[09-核心实体-Claude Desktop]] 类似，Cursor IDE 允许开发者在其环境中配置和使用像 [[07-核心实体-Neo4j Data Modeling MCP server]] 这样的工具。这意味着开发者可以在编码的同时，利用 AI 代理的能力来辅助完成 [[002-核心概念-Graph Data Modeling|图数据建模]]、代码生成和可视化等任务。

文章中提到了 Cursor IDE，是为了说明 [[001-核心概念-Model Context Protocol (MCP)|MCP]] 生态的广泛性，它不仅限于特定的聊天应用，也能集成到专业的开发工具中，为开发者提供更加无缝和强大的 AI 辅助编程体验。
