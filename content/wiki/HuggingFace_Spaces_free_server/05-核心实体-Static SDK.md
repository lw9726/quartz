Static SDK是Hugging Face Spaces中用于托管静态网站项目（如HTML、React、Vue等）的选项。
> 然后第三个的话主要运行一些静态的项目的。它也是完全免费的。
> 比如你可以在里面跑一些单页面的这个项目，或者轻量级的 Gradio，Transformer JS，React，或者是 Vue 等等。

Static SDK是[[02-核心实体-Hugging Face Spaces]]提供的第三种部署方式，专门用于托管静态前端项目。这对于那些需要为AI大模型搭建前端界面（如使用Chat UI）或展示纯前端应用的用户来说非常有用。与[[03-核心实体-Gradio SDK]]和[[04-核心实体-Docker SDK]]一样，使用Static SDK同样可以享受平台的[[009-核心流程-选择免费硬件套餐|免费硬件资源]]。
