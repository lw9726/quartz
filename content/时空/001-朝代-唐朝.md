# 001-朝代-唐朝

**核心概述:** 唐朝（618-907年）是中国历史上的黄金时代，也是中国古典诗歌发展的巅峰时期。唐诗以其丰富的内容、完美的艺术形式和深远的影响，成为中华文化的瑰宝。

**历史分期:**
- **初唐**（618-712）：诗歌创作的奠基期
- **盛唐**（713-765）：诗歌创作的黄金期  
- **中唐**（766-835）：诗歌创作的转变期
- **晚唐**（836-907）：诗歌创作的总结期

**文化特征:**
1. **开放包容**：海纳百川的文化胸襟
2. **繁荣昌盛**：经济发达，文化繁荣
3. **人才荟萃**：群星璀璨的文学时代
4. **国际交流**：丝绸之路促进文化交融

**重要历史事件:**
- [[时空/004-事件-安史之乱]]：唐朝由盛转衰的转折点

- [[时空/014-事件-开元盛世]]：唐玄宗前期的繁荣景象

**重要地点:**

**诗歌特色:**
1. **[[概念/033-概念-盛唐气象]]**：宏大的气势和开阔的胸襟
2. **[[概念/034-概念-格律完善]]**：近体诗格律体系的成熟
3. **[[概念/035-概念-题材丰富]]**：涵盖社会生活的各个方面
4. **[[概念/036-概念-风格多样]]**：浪漫主义与现实主义并存

**主要流派:**
- [[概念/001-概念-浪漫主义]]：以李白为代表
- [[概念/004-概念-现实主义]]：以杜甫为代表
- [[概念/037-概念-山水田园派]]：以王维、孟浩然为代表
- [[概念/038-概念-边塞诗派]]：以高适、岑参为代表

**文学成就:**
- **诗歌数量**：现存唐诗约5万首，诗人2300多位
- **艺术水平**：达到中国古典诗歌的最高峰
- **影响深远**：对后世文学创作影响深远，成为中华文化的重要组成部分

**社会背景:**
唐朝的政治稳定、经济繁荣、文化开放为诗歌创作提供了良好环境。[[概念/039-概念-科举制度]]的完善使文人地位提高，促进了诗歌创作的繁荣。

**后世影响:**
唐诗成为中国古典诗歌的标杆，"唐诗宋词"并称为中华文学的双峰。唐诗的艺术成就和精神内涵至今仍在滋养着中华民族的文化生活。
